WHAT IS THIS WEBSITE FOR?

I developed this site as a repository of vocabulary and demonstrations of concepts related to coding websites. You can click the links above to view information specific to either html or css. For example, below you will see a demonstration of how the 'Box Model' works.

Whenever you see one of these CodePen embeds, you can click the tabs marked HTML, CSS or RESULTS to toggle between the html source code, the stylesheet or the live preview. You can also click the 'Edit on CodePen' link to experiment with the code yourself. It is a very hands on way of learning and reviewing what we cover in class. I will be adding vocabulary and demonstrations as we progress through the semester.
